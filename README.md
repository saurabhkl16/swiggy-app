
# WELCOME TO SWIGGY APP
Simple Java CLI Application

## Description
This is a pure Java project that allows users to add and view users using a console menu.
It demonstrates object-oriented programming concepts such as classes, methods, and encapsulation.

## Features
- Add new users with name, mobile, and address
- View all users
- Login menu (simulated)
- Exit option

## How to Run
1. Open the project in IntelliJ IDEA.
2. Compile the project (Build → Build Project).
3. Run the `Main` class.
4. Follow the menu options in the console.

## Usage
- Choose `1` to add a user
- Choose `2` to open the login menu
- Choose `0` to exit
